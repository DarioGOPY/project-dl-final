### Objetivos del proyecto
Este proyecto tiene como objetivos:
- Implementar y comparar modelos concretos de cada familia (CNN, GAN, LSTM, Transformer) en tareas - representativas. 
- Preparar datos, entrenar, evaluar y explicar resultados. 
- Documentar el proceso en notebooks y un reporte profesional que sirva para entrevistas. 
- Practicar buenas prácticas: control de versiones, experiment tracking, reproducibilidad y visualización.
